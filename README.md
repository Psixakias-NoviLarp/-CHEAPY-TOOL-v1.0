## 🚀 Installation
```bash
git clone https://github.com/psixakias7/CheapyTool
cd CheapyTool
pip install -r requirements.txt
python main.py
```

## 📋 Requirements
- Python 3.8+
- Windows 10/11
- See requirements.txt
